# 命名

## 创建节点和附加命名空间

- c++ 节点名

```
// 程序中的init的节点名不包含命名空间（i.e '/.../'）
ros::init(argc, argv, "talker"); 
// launch中的name也不包含命名空间
<node pkg="talker" type="talker" name="talker"/>
// launch中给节点添加命名空间
<group ns="namespace1">  <!--group可同时给多个节点附上一个命名空间，ns则针对具体的一个，本部分可覆盖-->
	<node ns='namespace2' pkg="talker" type="talker" name="talker"/>
</group>
```



## 给主题添加命名空间

``` 
// 当创建节点有命名空间时
base_name类型的topic会附上节点的命名空间
// 当创建的句柄带有(~)时
base_name类型的topic除了附上节点的命名空间，还会附上节点名作为命名空间
```
