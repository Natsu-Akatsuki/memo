05. 监控
========

进程
----

htop(进程)
^^^^^^^^^^

一般查看当前用户下最占用cpu（P）和内存（M）的进程


* 命令行

.. prompt:: bash $,# auto

   # 只查看当前用户的进程
   $ htop -u $(whoami)
