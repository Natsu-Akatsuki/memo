`安装Node.js <https://nodejs.org/en/download/>`_
----------------------------------------------------

.. prompt:: bash $,# auto

   # apt下载npm(default: v10.19.0)
   $ sudo apt install nodejs npm
   # 升级npm
   $ npm install -g npm
   # 设置 淘宝镜像源
   $ npm config set registry https://registry.npm.taobao.org
   # 查看使用的镜像源
   $ npm config get registry
   # 查看已安装的npm包
   $ npm list
