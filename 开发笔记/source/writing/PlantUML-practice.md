# PlantUML-practice

## [安装](https://plantuml.com/zh/download)

```bash
$ wget -c https://github.com/plantuml/plantuml/releases/download/v1.2022.0/plantuml-1.2022.0.jar
# 安装java
$ sudo apt install openjdk-17-jre
```

## 导出SVG文件

```bash
$ java -jar plantuml.jar -tsvg .
```

- 多行表示：用 `:` 和 `;` 包围文字
